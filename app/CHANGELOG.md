# CHANGELOG.md

## [0.2.0] - 2026-03-07

### 🎮 Добавлено

#### Система мира
- **Куры спавнятся в каждом чанке** (5-15 на чанк)
- **Куры существуют постоянно** (не удаляются при уходе с экрана)
- **Счётчик "открытых" кур** (только из посещённых чанков)
- **Счётчик всех кур** (для отладки)

#### Рост мира
- **Трава растёт со временем** (проверка каждые 5 секунд)
- **Шанс роста травы** (2% за тик)
- **Трава может превратиться в лес** (0.5% шанс при 100% росте)
- **Грязь зарастает травой** (1% шанс за тик)

#### Конфигурация
- **GameConfig.kt** — централизованные настройки игры
- **Все параметры вынесены в один файл**:
  - Размер чанка (32 тайла)
  - Количество кур на чанк (5-15)
  - Время покоя курицы (50-150 кадров)
  - Дистанция ходьбы курицы (1-5 блоков)
  - Скорость роста травы (5000ms)
  - И другие параметры

#### Исправления графики
- **Убраны коричневые пиксели на траве** (увеличен шанс GRASS до 85%)
- **Куры спавнятся только на подходящих тайлах** (трава, песок)
- **Добавлена отрисовка травинок** (при росте >50%)

### 🔧 Исправлено

#### Критические ошибки
- **Бесконечная рекурсия** в World.kt (spawnChickensInChunk → getTile → generateChunk)
  - Решение: передача готового массива чанка вместо вызова getTile()
- **StackOverflowError** при генерации мира
  - Решение: разделение генерации тайлов и спавна существ

#### Синтаксические ошибки
- Удалены лишние пробелы в ключевых словах во всех файлах:
  - `Canva s` → `Canvas`
  - `& &` → `&&`
  - `fal se` → `false`
  - `sk inColor` → `skinColor`
  - `pixe lSize` → `pixelSize`
  - `Pain t()` → `Paint()`
  - `di rection` → `direction`
  - `chick ens` → `chickens`
  - `lateinit v ar` → `lateinit var`
  - `Canva s` → `Canvas`
  - `Styl e.FILL`

## [0.1.0] - 2026-03-02

### Добавлено
- Инициализация проекта "God Simulator"
- Базовая структура Android приложения
- Пиксельный человечек с видом сверху (детализированный)
    - Голова (3x3 пикселя)
    - Глаза с изменением направления взгляда
    - Шея, тело, руки, ноги
    - Штаны, ботинки, пояс
    - Пуговицы на теле
    - Анимация ходьбы (движение рук и ног)

- Игровой движок на SurfaceView
    - Плавный геймплей при 60 FPS
    - Система случайного движения
    - Отскок от границ экрана
    - Отображение FPS для отладки

- Пиксельное окружение
    - Зеленая трава с текстурой
    - Случайные цветочки на поле (красные, желтые, розовые)
