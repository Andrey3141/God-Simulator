# Changelog

## [0.1.0] - 2026-03-02

### Добавлено
- Инициализация проекта "God Simulator"
- Базовая структура Android приложения
- Пиксельный человечек с видом сверху (детализированный)
    - Голова (3x3 пикселя)
    - Глаза с изменением направления взгляда
    - Шея, тело, руки, ноги
    - Штаны, ботинки, пояс
    - Пуговицы на теле
    - Анимация ходьбы (движение рук и ног)

- Игровой движок на SurfaceView
    - Плавный геймплей при 60 FPS
    - Система случайного движения
    - Отскок от границ экрана
    - Отображение FPS для отладки

- Пиксельное окружение
    - Зеленая трава с текстурой
    - Случайные цветочки на поле (красные, желтые, розовые)

### Технические детали
- Настройка Gradle с Kotlin DSL
- Поддержка Android SDK 36
- Минимальная версия SDK: 24
- Использование корутин для игрового цикла
- Версия кода: 1
- Версия приложения: 1.0

### Зависимости
- AndroidX Core KTX 1.17.0
- AndroidX AppCompat 1.7.1
- Material Design 1.13.0
- AndroidX Activity 1.12.4
- AndroidX ConstraintLayout 2.2.1
- JUnit 4.13.2 для тестирования

### Известные проблемы
- Отсутствует управление (пока только случайное движение)
- Нет сохранения состояния игры
- FPS отображается в левом верхнем углу (будет убрано в релизной версии)
