## [v0.4.0] - 2026-03-08

### Added
- Система мира:
  - Текстуры для всех тайлов: трава, песок, снег, камень, вода, дерево, грязь
  - Дерево как отдельный тайл с текстурой tree.jpg
  - Пустыня использует текстуру песка (отдельный файл desert.jpg удалён)
- Поведение куриц:
  - Анимация поедания из chicken_eat.png (4 направления × 4 кадра)
  - Курицы могут есть траву (30% шанс при простое)
  - Трава превращается в грязь после поедания курицей
  - Параметры в GameConfig: CHICKEN_EAT_CHANCE = 0.3f, CHICKEN_EAT_DURATION_FRAMES = 60, CHICKEN_SPAWN_CHANCE = 0.03f
- Рост деревьев:
  - Базовый шанс: 0.5% за тик
  - Бонус от грязи: +6% за соседний тайл
  - Бонус от травы: +5% за соседний тайл
  - Бонус от песка: +0.1% за соседний тайл
  - Максимальный шанс: 100%
- Движение куриц:
  - Курицы не заходят на запрещённые тайлы: WATER, TREE, FOREST, STONE
  - Проверка каждого тайла на пути — курицы не проходят сквозь препятствия
  - Спавн только на разрешённых тайлах — проверка при генерации

## [v0.3.0] - 2026-03-07

### Added
- Система сохранений:
  - WorldSave.kt — сохранение мира в SharedPreferences (JSON)
  - Кнопка "💾 Сохранить" — ручное сохранение + включение автосохранения
  - Кнопка "🔄 Новый мир" — очистка сохранения и генерация заново
  - Автосохранение при выходе из игры (если включено)
- Ограничения для куриц:
  - Не спавнятся в воде, лесу, на камне, снегу, пустыне
  - Не заходят в воду и лес при движении
  - Предпочитают траву, песок, грязь
- Ориентация экрана:
  - Поддержка портрета и ландшафта (android:screenOrientation="sensor")
  - Адаптация камеры при повороте (onSizeChanged)
  - Адаптивный UI (размер текста/кнопок меняется)

### Fixed
- Критические ошибки:
  - Исправлена утечка корутин при перезапуске активности (scope?.cancel() + scope = null в surfaceDestroyed)

## [v0.2.0] - 2026-03-07

### Added
- Система мира:
  - Куры спавнятся в каждом чанке (5-15 на чанк)
  - Куры существуют постоянно (не удаляются при уходе с экрана)
  - Счётчик "открытых" кур (только из посещённых чанков)
  - Счётчик всех кур (для отладки)
- Рост мира:
  - Трава растёт со временем (проверка каждые 5 секунд)
  - Шанс роста травы (2% за тик)
  - Трава может превратиться в лес (0.5% шанс при 100% росте)
  - Грязь зарастает травой (1% шанс за тик)
- Конфигурация:
  - GameConfig.kt — централизованные настройки игры
  - Все параметры вынесены в один файл (размер чанка 32 тайла, количество кур на чанк 5-15, время покоя курицы 50-150 кадров, дистанция ходьбы курицы 1-5 блоков, скорость роста травы 5000ms и другие)
- Исправления графики:
  - Убраны коричневые пиксели на траве (увеличен шанс GRASS до 85%)
  - Куры спавнятся только на подходящих тайлах (трава, песок)
  - Добавлена отрисовка травинок (при росте >50%)

### Fixed
- Критические ошибки:
  - Исправлена бесконечная рекурсия в World.kt (spawnChickensInChunk → getTile → generateChunk) через передачу готового массива чанка вместо вызова getTile()
  - Исправлен StackOverflowError при генерации мира через разделение генерации тайлов и спавна существ
- Синтаксические ошибки (удалены лишние пробелы в ключевых словах во всех файлах):
  - Canva s → Canvas
  - & & → &&
  - fal se → false
  - sk inColor → skinColor
  - pixe lSize → pixelSize
  - Pain t() → Paint()
  - di rection → direction
  - chick ens → chickens
  - lateinit v ar → lateinit var
  - Styl e.FILL → Style.FILL
